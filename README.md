Ola Ride Data Analytics Dashboard

📌 Project Overview
This project analyzes Ola ride booking data to extract meaningful business insights related to bookings, revenue, cancellations, customer behavior, and vehicle performance.
An interactive dashboard has been developed using Power BI to visualize key metrics and trends for better decision-making.
🎯 Objectives
Analyze overall booking and revenue performance
Understand ride success and cancellation patterns
Identify key reasons for ride cancellations
Compare performance across vehicle types
Evaluate customer and driver ratings
Analyze revenue trends and payment methods
